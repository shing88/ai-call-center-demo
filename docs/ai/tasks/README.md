# タスク指示

このディレクトリには、エージェント向けの短く実行可能なタスク指示を保存する。

各タスクは1つのプルリクエストに収まる大きさにし、完了に必要なファイルとコンテキストだけを指定する。

## ルール

- アクティブなタスクは`docs/ai/context/ACTIVE_TASK.md`から参照される状態に保つ。
- `docs/ai/inbox/pro-instructions/`にあるGPT Proドラフトを直接実行しない。
- 計画ドラフトを実行可能なタスクへ移す前に、現在のコード、テスト、正本順序、未解決の質問を確認する。
- 追加で利用したコンテキストはPR本文に記録する。
