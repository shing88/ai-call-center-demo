# CCNet Executive Demo Scenario

最終更新: 2026-05-20

このシナリオは、CCNet株式会社の公開HPを確認したうえで作る役員デモ用の架空シナリオ。実在のお客様、契約、問い合わせ履歴、障害情報は使わない。

## 公開HPから拾う会社フィット

- 会社概要: CCNetは愛知・岐阜・三重の一部エリアで、有線テレビジョン放送、データ通信、電気通信などを事業としている。
  参照: https://www.ccnw.co.jp/company/profile.html
- サービス: インターネット、テレビ、固定電話、動画配信、ケーブルスマホ、電気・ガスをサービス案内に持つ。
  参照: https://www.ccnw.co.jp/service/
- インターネット: 高速インターネット、Wi-Fi標準提供、マカフィーセキュリティ、安心サポートを訴求している。
  参照: https://www.ccnw.co.jp/service/internet/
- 10G / Wi-Fi: CCNet光10Gではテレワーク、動画、複数デバイス、Wi-Fi 6対応ルーター、利用環境や提供可否の条件が重要になる。
  参照: https://www.ccnw.co.jp/campaign/10g/
- 料金・オプション: 戸建て向けインターネット料金では10G、1G、300M、30M、光ライトが案内され、メッシュWi-Fiは2台セット月額990円、追加1台330円のオプションとして案内されている。
  参照: https://www.ccnw.co.jp/price/house/internet.html
- 約款・重説: 契約約款、戸建て向け注意事項・重要事項、初期契約解除制度は、契約状態、変更、解約、契約期間、解除料、工事費残債、貸与物返却、初期契約解除の確認観点として使う。
  参照: https://www.ccnw.co.jp/provision/ と https://www.ccnw.co.jp/simulation/pdf/terms/net.pdf?260401= と https://www.ccnw.co.jp/simulation/pdf/terms/house.pdf?2411= と https://www.ccnw.co.jp/provision/pdf/other-cancel.pdf?2404=
- 地域情報: CCNet12と安全・安心123チャンネルで、地域情報、防災・防犯、道路・河川カメラなどを扱う。
  参照: https://www.ccnw.co.jp/ccnet_channel/ と https://www.ccnw.co.jp/service/tv/ch123/
- サポート導線: 問い合わせ前にFAQや操作・設定マニュアル、加入者向けにはマイページが案内され、インターネット・固定電話のサポート窓口もある。
  参照: https://www.ccnw.co.jp/contact/ と https://www.ccnw.co.jp/users/support/

## 役員デモシナリオ

### タイトル

10G / Wi-Fiサポートと地域安全情報の横断ハンドオフ

### 架空の問い合わせ

春日井市エリアの加入者という設定の架空顧客が、テレワーク会議中に自宅Wi-Fiが不安定になることを相談する。あわせて、CCNet光10Gへの変更可否、Wi-Fi 6ルーターやあんしんサポート、台風接近時に安全・安心123チャンネルで道路・河川情報を確認できるかを聞いている。

### 架空顧客モック

- デモ顧客ID: `customer_ccnet_2001`
- エリア: 春日井市 / 戸建て。
- 現在のサービス: CCNet光1G おとく割 + テレビ + ケーブルライン + メッシュWi-Fi 2台。
- 契約確認観点: 36ヶ月契約、10G変更可否、提供エリア、工事要否、料金差分、貸与機器返却、初期契約解除制度。
- ガードレール: 契約番号、電話番号、番地、実在問い合わせ履歴は持たない。本人確認前は契約状態、料金、解約料、10G提供可否を断定しない。

### デモで見せる流れ

1. `Executive demo brief`で、これは公開HPに合わせた架空シナリオであり、外部送信・永続保存・本番接続はないと先に説明する。
2. `Fictional customer mockup`で、架空顧客ID、エリア、現在サービス、本人未確認状態を確認する。
3. `Evidence candidates`で、10G / Wi-Fi、メッシュWi-Fi、約款・重説、サポート導線、地域情報チャンネルの根拠候補を確認する。
4. `Policy guard`で、本人確認前はコース変更、提供可否、料金、契約状態、解約料、初期契約解除適用可否を断定しないことを示す。
5. `Operator note`では、問い合わせ要点をbrowser-onlyの未送信候補として残し、送信済み・保存済みではないことを示す。
6. `Fallback rehearsal`では、通信や外部AI APIが使えない場合も、固定済みシナリオとpolicy guardで説明を続けられることを示す。

### 回答方針

- 一般案内として、Wi-Fiや利用環境、サポート導線、地域情報チャンネルを説明する。
- 10Gの提供可否、工事可否、料金、契約変更日、解約料、工事費残債、貸与物返却は本人確認とエリア・設備確認後に案内する。
- 不具合や障害、補償、契約変更の確定判断は、必要に応じて人の担当者へ引き継ぐ。
