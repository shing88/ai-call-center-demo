# CCNet Executive Demo Scenario

最終更新: 2026-05-20

このシナリオは、CCNet株式会社の公開HPを確認したうえで作る役員デモ用の架空シナリオ。実在のお客様、契約、問い合わせ履歴、障害情報は使わない。

## 公開HPから拾う会社フィット

- 会社概要: CCNetは愛知・岐阜・三重の一部エリアで、有線テレビジョン放送、データ通信、電気通信などを事業としている。
  参照: https://www.ccnw.co.jp/company/profile.html
- サービス: インターネット、テレビ、固定電話、動画配信、ケーブルスマホ、電気・ガスをサービス案内に持つ。
  参照: https://www.ccnw.co.jp/service/
- インターネット: 高速インターネット、Wi-Fi標準提供、マカフィーセキュリティ、安心サポートを訴求している。
  参照: https://www.ccnw.co.jp/service/internet/
- 10G / Wi-Fi: CCNet光10Gではテレワーク、動画、複数デバイス、Wi-Fi 6対応ルーター、利用環境や提供可否の条件が重要になる。
  参照: https://www.ccnw.co.jp/campaign/10g/
- 地域情報: CCNet12と安全・安心123チャンネルで、地域情報、防災・防犯、道路・河川カメラなどを扱う。
  参照: https://www.ccnw.co.jp/ccnet_channel/ と https://www.ccnw.co.jp/service/tv/ch123/
- サポート導線: 問い合わせ前にFAQや操作・設定マニュアル、加入者向けにはマイページが案内され、インターネット・固定電話のサポート窓口もある。
  参照: https://www.ccnw.co.jp/contact/ と https://www.ccnw.co.jp/users/support/

## 役員デモシナリオ

### タイトル

10G / Wi-Fiサポートと地域安全情報の横断ハンドオフ

### 架空の問い合わせ

春日井市エリアの加入者という設定の架空顧客が、テレワーク会議中に自宅Wi-Fiが不安定になることを相談する。あわせて、CCNet光10Gへの変更可否、Wi-Fi 6ルーターやあんしんサポート、台風接近時に安全・安心123チャンネルで道路・河川情報を確認できるかを聞いている。

### デモで見せる流れ

1. `Executive demo brief`で、これは公開HPに合わせた架空シナリオであり、外部送信・永続保存・本番接続はないと先に説明する。
2. `Evidence candidates`で、10G / Wi-Fi、サポート導線、地域情報チャンネルの根拠候補を確認する。
3. `Policy guard`で、本人確認前はコース変更、提供可否、料金、契約状態を断定しないことを示す。
4. `Operator note`では、問い合わせ要点をbrowser-onlyの未送信候補として残し、送信済み・保存済みではないことを示す。
5. `Fallback rehearsal`では、通信や外部AI APIが使えない場合も、固定済みシナリオとpolicy guardで説明を続けられることを示す。

### 回答方針

- 一般案内として、Wi-Fiや利用環境、サポート導線、地域情報チャンネルを説明する。
- 10Gの提供可否、工事可否、料金、契約変更日は本人確認とエリア・設備確認後に案内する。
- 不具合や障害、補償、契約変更の確定判断は、必要に応じて人の担当者へ引き継ぐ。

## 禁止する言い方

- 実在顧客の契約情報を参照した。
- CCNetへ外部AI APIで問い合わせ済み。
- 顧客へ回答を送信済み。
- Operator noteをDBや履歴へ保存済み。
- 10G提供、速度、工事日、料金、補償をこの画面だけで保証できる。
